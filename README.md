## Overview
A full-stack MERN application to generate AI-powered thumbnails online using the Google Gemini API.

## Tech Stack
- Frontend: React.js, HTML5, CSS3, Material UI
- Backend: Node.js, Express.js, RESTful APIs, JWT Authentication
- Database: MongoDB
- AI API: Google Gemini API
- Tools: Git, GitHub, Postman

## Planned Features
- User authentication (Sign up / Login)
- AI-based thumbnail generation
- Download & save thumbnails
- Responsive UI
- Cloud deployment
